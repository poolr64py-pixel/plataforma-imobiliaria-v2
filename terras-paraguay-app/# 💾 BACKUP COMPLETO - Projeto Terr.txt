### **Desenvolvimento Local:**
```bash
# 1. Clonar/criar projeto
npx create-react-app terras-paraguay
cd terras-paraguay

# 2. Instalar dependências
npm install lucide-react react-router-dom

# 3. Copiar arquivos do backup
# - App-TerrasParaguay.js → src/
# - terrasParaguayData.js → src/data/
# - Componentes → src/components/
# - Utils → src/utils/

# 4. Iniciar desenvolvimento
npm start
```

### **Deploy Produção:**
```bash
# Opção 1: Netlify
npm run build
# Upload pasta build/ para Netlify

# Opção 2: Vercel  
vercel --prod

# Opção 3: Subdomínio
# Configurar DNS: imoveis.terrasnoparaguay.com
# Apontar para servidor/CDN
```
